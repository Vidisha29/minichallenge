# 🍱 Kanso

> **Your day, thoughtfully cooked.**

Kanso is an AI-powered cooking concierge that creates a personalized daily meal plan based on your schedule, budget, dietary preferences, and available ingredients. Inspired by Japanese minimalism, Kanso transforms meal planning into a calm, effortless experience while helping users save time, reduce food waste, and stay within budget.

---

## 🌟 Why Kanso?

Most meal planners recommend recipes.

**Kanso plans your day.**

Instead of asking *"What do you want to eat?"*, Kanso understands:

- 🕒 Your schedule
- 🥗 Your dietary preferences
- 💰 Your budget
- 🧺 Ingredients already available at home
- 🍳 Your cooking goals

It then creates a complete, optimized cooking plan powered by AI.

---

# ✨ Features

## 🤖 AI Meal Planning
Generate personalized Breakfast, Lunch, and Dinner plans based on your daily routine.

---

## 📷 Pantry Scanner

Upload a photo of your pantry or manually enter available ingredients.

AI automatically identifies ingredients and uses them while planning meals.

---

## 🛒 Smart Grocery List

Automatically generates:

- Ingredients you already have
- Ingredients to buy
- Organized shopping checklist

---

## 🔄 Intelligent Ingredient Substitutions

If an ingredient isn't available, AI recommends alternatives based on:

- Taste similarity
- Budget
- Dietary preference
- Availability

---

## 💰 Budget Optimizer

Kanso estimates:

- Meal cost
- Grocery cost
- Money saved
- Budget utilization

---

## 🧠 AI Reasoning

Every recommendation includes an explanation.

Example:

> "This meal was selected because you chose a busy workday, a ₹500 budget, and already have eggs and vegetables available."

---

## 🌱 Food Waste Reduction

Kanso prioritizes ingredients already available at home before suggesting new purchases, helping reduce unnecessary food waste.

---

# 🎯 Problem Statement

Planning meals every day can be:

- Time-consuming
- Expensive
- Repetitive
- Wasteful

Most existing meal planners ignore real-life constraints like schedules, budgets, and pantry inventory.

Kanso solves this by combining AI with practical daily planning.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/yourusername/kanso.git
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```
